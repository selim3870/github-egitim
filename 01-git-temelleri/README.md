# 1. Gün - Git Temelleri

## 🎯 Bugün ne öğreneceksiniz?

- Git nedir, neden kullanılır?
- İlk Git deposunu oluşturma (`git init`)
- Dosya ekleme (`git add`) ve commit yapma (`git commit`)
- Değişiklikleri takip etme (`git status`, `git log`)

## 📋 Ön koşul

- Git kurulu olmalı ([git-scm.com](https://git-scm.com) adresinden indirin)
