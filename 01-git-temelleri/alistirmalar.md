# 1. Gün - Alıştırmalar

## Alıştırma 1: İlk repo
1. Masaüstünde `git-egitim` klasörü oluştur
2. `cd git-egitim`
3. `git init`

## Alıştırma 2: İlk commit
1. `selam.txt` oluştur, içine "Merhaba Git" yaz
2. `git add selam.txt`
3. `git commit -m "İlk commit"`

## Alıştırma 3: Değişiklik yap
1. `selam.txt`'ye "İkinci satır" ekle
2. `git add .`
3. `git commit -m "İkinci satır eklendi"`

## ✅ Başarı kriterleri
- [ ] `git init` çalıştı
- [ ] `git add` çalıştı
- [ ] `git commit` çalıştı
- [ ] `git log` ile commit'ler görüldü
