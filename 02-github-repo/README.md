# 2. Gün - GitHub'a Bağlanma

## 🎯 Bugün ne öğreneceksiniz?

- Yerel repoyu GitHub'a bağlama (`git remote`)
- Kodları GitHub'a gönderme (`git push`)
- GitHub'dan güncelleme çekme (`git pull`)
- Bir repoyu kopyalama (`git clone`)

## 📋 Ön koşul

- 1. gün tamamlanmış olmalı
- GitHub hesabınız olmalı ([github.com](https://github.com) ücretsiz)
