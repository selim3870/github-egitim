# 🚀 Sıfırdan GitHub Uzmanlığı Eğitimi

Bu repo, **Git ve GitHub'ı hiç bilmeyenler** için hazırlanmış **ücretsiz ve adım adım** bir eğitim setidir.

## 📖 Bu eğitim kimler için?

- Git nedir bilmeyenler
- Takım çalışmasında GitHub kullanmak isteyenler

## 🗺️ Nasıl kullanılır?

1. Müfredatı takip edin.
2. Her gün **1 modül** işleyin.
3. Alıştırmaları **yapmadan** geçmeyin.
