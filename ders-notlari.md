# 🚀 Sıfırdan GitHub Uzmanlığı Eğitimi

Bu repo, **Git ve GitHub'ı hiç bilmeyenler** için hazırlanmış **ücretsiz ve adım adım** bir eğitim setidir.

## 📖 Bu eğitim kimler için?

- Git nedir bilmeyenler
- "commit", "branch", "pull request" duymuş ama ne olduğunu anlamayanlar
- Takım çalışmasında GitHub kullanmak isteyen yazılımcılar
- Kendi kendine öğrenmek isteyen herkes

## 🗺️ Nasıl kullanılır?

1. Aşağıdaki **müfredatı** takip edin.
2. Her gün **1 modül** işleyin (acele etmeyin).
3. Her modülde sırayla şu dosyaları okuyun:
   - `README.md` → Bu modülde ne öğreneceğinizi anlatır
   - `komutlar.md` → Komutları ve ne işe yaradıklarını açıklar
   - `alistirmalar.md` → Pratik yapmanız için alıştırmalar
4. Alıştırmaları **yapmadan** bir sonraki modüle geçmeyin.

## 📅 Müfredat (9 Gün)

| Gün | Konu | Ne öğreneceksin? |
|-----|------|------------------|
| 1 | Git Temelleri | `git init`, `add`, `commit`, `status`, `log` |
| 2 | GitHub'a Bağlanma | `remote`, `push`, `pull`, `clone` |
| 3 | Branch ve Pull Request | `branch`, `checkout`, `merge`, PR açmak |
| 4 | Fork ve Upstream | Başkasının projesine katkı yapmak |
| 5 | Code Review | PR inceleme, yorum yapma, approve |
| 6 | GitHub Actions | Otomatik test ve iş akışları |
| 7 | Branch Protection | main branch'ini koruma kuralları |
| 8 | GitHub CLI | `gh` komutu ile terminalden GitHub |
| 9 | İleri Seviye | Rebase, cherry-pick, reflog |

## 🎯 Hedef

Bu eğitimin sonunda:
- ✅ Git ile bir projeyi versiyonlayabileceksiniz
- ✅ GitHub'da repo oluşturup yönetebileceksiniz
- ✅ Branch açıp Pull Request gönderebileceksiniz
- ✅ Başkasının projesine katkı yapabileceksiniz
- ✅ GitHub Actions ile otomasyon kurabileceksiniz
